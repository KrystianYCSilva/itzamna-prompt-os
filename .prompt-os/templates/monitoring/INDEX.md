## ðŸ“Š REPORT TYPES

### Evolution Report
Tracks feature progression:
- Plan vs. actual timeline
- Phase completion percentage
- Quality checkpoints
- Blockers and resolutions

### Gap Detection Report
Identifies missing capabilities:
- Capability gap name
- Impact assessment
- Estimated effort to close
- Priority/tier

### Rejection Analysis Report
Analyzes decision rejections:
- What was rejected
- Rejection reason
- Feedback from reviewer
- Path forward

### Self-Critique Metrics
Scores artifact quality:
- Completeness (0-100)
- Clarity (0-100)
- Correctness (0-100)
- Best practices (0-100)
- Overall score (0-100)
